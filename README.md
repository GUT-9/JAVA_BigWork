## 环境配置
### 数据库配置
数据库使用SQL  server本地部署，使用项目前请运行**SQL server**的服务。
使用管理员用户sa，用户密码123进行登录，所以请确保这个用户可以登录并设置了该密码。

数据库无需手动配置，如果没有创建数据库会自动创建，功能集成在类`DatabaseConfig`类中,但请务必完成上面操作。
